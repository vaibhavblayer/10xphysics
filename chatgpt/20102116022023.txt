Q. What are the top three books to learn machine learning using python?

1. Machine Learning with Python Cookbook: Practical Solutions from Preprocessing to Deep Learning by Chris Albon 
2. Python Machine Learning: Machine Learning and Deep Learning with Python, scikit-learn, and TensorFlow, 2nd Edition by Sebastian Raschka 
3. Hands-On Machine Learning with Scikit-Learn and TensorFlow: Concepts, Tools, and Techniques to Build Intelligent Systems by Aurélien Géron
