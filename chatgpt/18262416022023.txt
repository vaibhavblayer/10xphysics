Q. How to update database using python in sqlite3 using sqlite3 module?

You can use the execute() method to execute SQL queries in SQLite3.

For example, if you want to update a record in a table called 'customers' you can use the following code:

import sqlite3

# Connect to the database
conn = sqlite3.connect('example.db')

# Create a cursor
cursor = conn.cursor()

# Execute the query 
query = 'UPDATE customers SET name = "John Doe" WHERE id = 1;'
cursor.execute(query)

# Commit the changes
conn.commit()

# Close the connection
conn.close()
