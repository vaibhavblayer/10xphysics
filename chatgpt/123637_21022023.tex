\textbf{Definition:}
A function $f$ is a mapping from a set $X$ to a set $Y$.

\textbf{Example:}
Let $X = \{1, 2, 3\}$ and $Y = \{a, b, c\}$. Define a function $f$ such that $f(1) = a$, $f(2) = b$, and $f(3) = c$.

We can call this function $f$ as follows:

$f(1) = a$

$f(2) = b$

$f(3) = c$
